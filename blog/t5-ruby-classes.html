Ruby Class
Class in Ruby is like objects in our world. It carries data, or characteristics. If we are a wine maker, we make wine, and we can make different kinds of wine – Cabernet Sauvignon, Merlot, Sauvignon Blanc. Each kind of wine has different taste, color, and best food to pair with. Let me demonstrate the class concept with this example.
Like defining a method, the first step to create a class in ruby is to name it. See how the class name is different than method names:
Class Wine
end
Cabernet Sauvignon is our first wine! This is how we can call it out:
wine_number = Wine.new(“wine_name”, “taste”, “color”, “pairing_food”)
But before we start to make our first wine, we know we want to make more than one type of wine. We need to tell ruby that each time we make a new wine with “Wine.new”, we are trying to make a different kind of wine. So how do we do that? It’s easy.
Class Wine
  def initialize(wine_name)
  @wine_name = wine_name
end
The initialize method brings up the concept “instance variable” which tells ruby each time we call out “Wine.new”, we are creating an instance of such class. After adding the initialize method, now we can proceed with our 3 kinds of wine with their ingredient and :
wine_1 = Wine.new(“cabernet_sauvignon” , “rich currant”, “red”, “red meat”)
wine_2 = Wine.new(“merlot”, “smooth black-cherry and herbal”, “red”, “any food”)
wine_3 = Wine.new(“sauvignon_blanc”, “sour fruit”, “white”, “seafood”)
Now we have 3 kinds of wine with their descriptions!
Let’s move on to the next step to make instance method inside class that will return each instance’s description upon calling:
Class Wine
   def initialize(wine_name)
     @wine_name = wine_name
   end

   def taste
     @taste
   end

   def color
     @color
   end

   def pairing_food
     @pairing_food
   end
end
The @ symbol in front of variable meaning it’s an instance variable. If I input
wine_1.pairing_food
Ruby will know I’m referring to the instance variable of wine_1, which is “cabernet sauvignon” and return its pairing food “red meat”
